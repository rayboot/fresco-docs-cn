类库发布到了Maven中央库:

## Gradle:

```groovy
dependencies {
  compile 'com.facebook.fresco:fresco:0.4.0+'
}
```

## Maven:

```xml
<dependency>
  <groupId>com.facebook.fresco</groupId>
  <artifactId>fresco</artifactId>
  <version>LATEST</version>
</dependency>
```
